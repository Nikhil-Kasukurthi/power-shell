# Power-shell
A collection of utility shell scripts to make a developer's job easy

##How to Install
####Clone this repository locally or download it as zip.

For Android Studio
```
sh android-studio.sh
```
___
For LAMP stack
```
sh lamp-stack-14-04.sh
```
___
For Python - Tornado 
```
sh python-tornado.sh
```
___
